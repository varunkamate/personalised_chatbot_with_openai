# personalised_chatbot_with_openai
This repository contains two different implementations of a chatbot, demonstrating both a simple rule-based approach and a sophisticated AI-powered solution using the OpenAI API.

The project requires an active OpenAI API key, which must be stored in a .env file.

API Key: The project utilizes a .env file (which is tracked in the directory structure but should be excluded from Git using .gitignore) to securely load the OPENAI_API_KEY.

Dependencies: All required packages are listed in requirements.txt.

Execution:

To run the Streamlit Web App: streamlit run app.py

To run the Terminal OpenAI Chatbot: python chatbot_with_openai.py

To run the Terminal NLTK Chatbot: python chatbot.py
